## 📜 Project Description

This project presents a detailed audit of Git as an open-source software. It covers its origin, licensing model, ethical considerations, Linux footprint, ecosystem, and comparison with proprietary tools.

---

## 💻 Scripts Included

### 1. System Identity Report

Displays system information such as kernel version, user, uptime, and operating system details.

### 2. FOSS Package Inspector

Checks whether Git is installed on the system and displays its version and basic details.

### 3. Disk and Permission Auditor

Analyzes important system directories and reports their size and permissions.

### 4. Log File Analyzer

Reads a log file, counts occurrences of a keyword (like errors), and displays matching lines.

### 5. Open Source Manifesto Generator

Generates a personalized open-source philosophy statement based on user input.

---

## ⚙️ How to Run

Make scripts executable:

```bash
chmod +x *.sh
```

Run any script:

```bash
./script1.sh
```

Run log analyzer:

```bash
./script4.sh /var/log/syslog error
```
