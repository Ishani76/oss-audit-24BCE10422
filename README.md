## Project Description
This project is an audit of Git, an open-source version control system. It includes analysis of its origin, license, ethical aspects, Linux footprint, ecosystem, and comparison with proprietary software.

## Scripts Included

### Script 1 — System Identity Report
Displays system information like kernel, user, uptime, and OS.

### Script 2 — FOSS Package Inspector
Checks if Git is installed and shows version.

### Script 3 — Disk and Permission Auditor
Analyzes system directories and permissions.

### Script 4 — Log File Analyzer
Counts occurrences of a keyword in a log file.

### Script 5 — Open Source Manifesto Generator
Generates a personalized open-source statement.
